# 月の満ち欠け・日食月食シミュレーター

ブラウザ上で太陽・地球・月の位置関係を操作しながら、**月の満ち欠け**と**日食・月食の発生条件**を同時に視覚的に理解できる Web 教材です。高校地学・地学基礎の定番単元を、平面図ではなく実際の天体配置と対応づけて学べます。

## 主な機能

- 月の公転角スライダー / 経過日数スライダー / 再生・一時停止・速度変更
- 俯瞰ビュー (Sun / Earth / Moon と本影・半影の可視化)
- 地球視点ビュー (太陽光による正確な明暗、月相名・月齢・位相角・照度表示)
- 白道傾斜 (5.14°) の ON / OFF 切替
- 新月・上弦・満月・下弦および次回日食・次回月食への一発移動
- 食判定: 皆既 / 金環 / 部分 / 半影月食
- 学習パネルの状態依存解説 & クイズ 3 種 (月相名・配置から月相・食の条件)
- 日本語 / English 切替、ライト / ダーク切替、レスポンシブ

## 技術

- Vanilla JavaScript (ES modules) + HTML + CSS + Canvas 2D
- ビルド不要。そのまま GitHub Pages で配信できます
- `.github/workflows/deploy.yml` によるデプロイ

## ローカル実行

任意の静的サーバで起動してください。

```sh
python3 -m http.server 8080
# または
npx serve .
```

ブラウザで `http://localhost:8080/` を開きます。

## 注意

教育用途のシミュレータであり、数値は近似、距離は視認性のために圧縮表示しています。実測・予報用途では使用しないでください。
